# DriverLog — โพสต์ Facebook (ภาษาไทย)

---

🚗 แนะนำ **DriverLog** — แอปบันทึกรายได้สำหรับคนขับรับจ้าง ใช้ฟรี!

เลิกจดรายได้ในสมุดได้แล้ว 📓 DriverLog ช่วยคุณบันทึกทุกกะการขับ
แล้วบอกได้ทันทีว่า **"กะนี้คุ้มไหม?"** 💰

✨ **ฟีเจอร์เด่น**
• บันทึกรอบขับง่าย ๆ — เลือกผู้ให้บริการ (Grab, Lineman, Bolt, Shopee, แท็กซี่) และประเภทงาน (รถ / มอเตอร์ไซค์ / อาหาร / ส่งของ)
• คำนวณ **รายได้สุทธิ** ให้อัตโนมัติ (รายได้ + ทิป − ต้นทุน)
• **"กะนี้คุ้มไหม?"** — ดูอัตรากำไร กำไรต่อชั่วโมง และจุดคุ้มทุน
• สถิติครบ — แนวโน้มรายได้ วันที่ทำเงินสูงสุด และช่วงเวลาที่ดีที่สุด
• เปรียบเทียบกับช่วงก่อน — วันนี้เทียบเมื่อวาน / สัปดาห์นี้เทียบสัปดาห์ก่อน
• รองรับ **กะกลางคืนข้ามวัน** ⏰
• สลับ **ไทย/อังกฤษ** 🌏 มี **โหมดมืด** 🌙 และใช้ได้แม้ **ออฟไลน์**
• ข้อมูลเป็นส่วนตัว เก็บไว้ในเครื่องคุณ 🔒

🗺️ **กำลังจะมา**
• ซิงก์ข้อมูลข้ามอุปกรณ์ (มือถือ + คอมพิวเตอร์)
• แอปบน Google Play
• สรุปรายได้รายสัปดาห์ + แจ้งเตือนก่อนเริ่มกะ
• รองรับงานกิ๊กอื่น ๆ นอกเหนือจากคนขับ
